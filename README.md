# RNN_challenge

В соревновании требовалось предсказать два анонимизированных финансовых индикатора основе последовательности таких же анонимизированных данных о котировках некоторого финансового актива/активов. 

Что в репозитории:
- в файле solution.py лежит основной код запуска модели, который нужно было сдавать в проверяющую систему вместе с самой моделью, веса которой должны хранитьсяв файле формата onnx. 
- В файле final_pipeline модель написан основной pipeline обучения модели, в конце модель импортируется в виде файла в формате onnx.
- Файл utils содержит полезные функции и класс datapoint, был выдан разработчиками соревнования и нужен для корректной работы solution.py, его нельзя было менять. 
- в папке data лежат тестовый и обучающий датасеты.
